 Q. What is Package.json ?

 package.json is a configuration file used in projects that use Node.js. It contains metadata about the project 
 and manages dependencies, scripts, and settings.
 example:
 {
  "name": "my-app",
  "version": "1.0.0",
  "description": "My first Node.js app",
  "main": "index.js",
  "scripts": {
    "start": "node index.js",
    "test": "echo \"No tests yet\""
  },
  "author": "Your Name",
  "license": "ISC",
  "dependencies": {
    "express": "^4.18.2"
  }
}

name → Project name
version → Current version
description → Short info about the project
main → Entry file
scripts → Commands you can run (npm start, etc.)
dependencies → Libraries your project needs
devDependencies → Tools needed only for development


Q what is node modules ?
In Node.js, node_modules is a folder that stores all the packages (libraries) your project depends on.

example: express, body-parser, braces etc.
