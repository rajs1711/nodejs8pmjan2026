COMPONENT
============
In nextjs/ react js a COMPONENT' is a reusable piece of UI built with React , 
Componnent helps you organize
your coe into smaller maintainable parts.

forexample:   form component , button compoent , navbar compoents 

In react we have two types of comoponent:
  1. Functional Component 
  2. Class Component (not used now ) now works which is done by class compoent handles by hooks

  for Example:

    return <button>Click Me</button>
